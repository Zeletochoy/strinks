Strinks - Beer Cost-Performance Monitoring
==========================================

Stonks + Drinks = ~terrible name idea~ Strinks

# Setup


* Get python 3.6+ and pip
* `pip install .`

## Formatting

To enable the automatic formatting git hook run `git config --local core.hooksPath .githooks/`
